# Памятка по Командам и Запуску Проекта ВЛЮБВИ

**Актуальная ссылка на проект:** [https://ubiquitous-monstera-97dbfe.netlify.app/](https://ubiquitous-monstera-97dbfe.netlify.app/)

Этот документ содержит все необходимые команды для ручного управления серверами, туннелями и деплоем на Netlify. Вы можете копировать их в терминал (PowerShell / CMD) по одной или пакетами, разделёнными точкой с запятой (;), как вы и просили.

---

## 1. Запуск Локального Бэкенда (FastAPI)
Откройте новый терминал и выполните (копируйте всю строку целиком):
```bat
cd "backend" ; .\venv\Scripts\activate ; uvicorn app.main:app --host 0.0.0.0 --port 8000
```
Если нужно будет перезапустить бэкенд, просто нажмите `Ctrl+C` и повторите эту команду.

## 2. Запуск Локального Туннеля для Бэкенда (API)
Чтобы фронтенд с Netlify мог достучаться до вашего локального компьютера, нужен туннель:
```bat
cd "backend" ; npx localtunnel --port 8000 --subdomain vlubvi-api
```
*(Ваш API теперь всегда будет доступен по адресу `https://vlubvi-api.loca.lt`)*

## 3. Запуск Телеграм-бота
Локальный запуск бота, который генерирует кнопку для MiniApp:
```bat
cd "backend" ; .\venv\Scripts\activate ; python bot.py
```

## 4. Сборка и Деплой Frontend на Netlify
Netlify решает проблему с 400 ошибками и экранами-заглушками при открытии внутри Telegram!

**Шаг 4.1. Установка Netlify CLI (один раз)**
```bat
npm install -g netlify-cli
```

**Шаг 4.2. Сборка (Build) фронтенда для продакшена**
```bat
cd "d:\Vpoiske App\frontend" ; npm run build
```

**Шаг 4.3. Деплой на Netlify**
```bat
cd "d:\Vpoiske App\frontend" ; npx netlify deploy --prod --dir=dist
```
Во время первого деплоя терминал может спросить, хотите ли вы привязать существующий сайт или создать новый `Create & configure a new site`. Выбирайте создание нового. Авторизуйтесь в браузере, и он выдаст вам готовую публичную HTTPS-ссылку (например: `https://vlubvi-app.netlify.app`), которая ИДЕАЛЬНО работает в Telegram!

После получения ссылки от Netlify вы должны:
1. Зайти в `backend/.env`
2. Поменять `WEB_APP_URL` на вашу новую ссылку от Netlify! (например: `WEB_APP_URL=https://vlubvi-app.netlify.app/onboarding`)
3. Перезапустить `bot.py`

## 5. Подключение ИИ Агента (Grok / Gemini / GPT)
Для генерации Human Design вопросов и "живого" интервью нужен бесплатный API ключ. Самый мощный бесплатный вариант на сегодня — **Groq (Llama 3)** или **Google Gemini**.

**Где взять ключ бесплатно:**
* Groq: Зайдите на `console.groq.com`, зарегистрируйтесь, выпустите API Key (он невероятно быстрый).
* Либо используйте бесплатную библиотеку `g4f` (которая уже встроена мною в проект в качестве запасного плана без ключей, но она может изредка тормозить).

**Текущая реализация ИИ:**
Я перевел код бэкенда на использование бесплатной генеративной Python библиотеки для тестирования (или заменю на библиотеку для Groq). Команды установки:
```bat
cd "d:\Vpoiske App\backend" ; .\venv\Scripts\activate ; pip install g4f
```
