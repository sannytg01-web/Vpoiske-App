1. «Карта теневых паттернов» — шеаринг-магнитЭто самая вирусная фича, которой нет у конкурентов. После интервью показывать пользователю красивую персонализированную карточку с его shadow_patterns и attachment_style, оформленную в стиле приложения (glassmorphism + тёплые цвета). Например: «Твоя тень: самопожертвование + страх быть брошенным. В отношениях ты — якорь, который боится, что корабль уйдёт». Добавить кнопку «Поделиться в Telegram/Stories» — люди будут сами распространять контент, потому что это резонирует эмоционально. Это встроенный виральный механизм.

2. AI-тренажёр первого сообщенияПрямое решение боли №1 по вашему исследованию (страх первого сообщения). После матча пользователь нажимает «Помоги написать» — AI генерирует 3 варианта первого сообщения на основе обоих психологических профилей. Например: если партнёр anxious attachment + ценит глубину, AI предложит вопрос про смыслы, а не комплимент. Это не просто фишка — это продукт внутри продукта. Подобного нет ни у Тиндера, ни у Hinge.

3. «Compatibility Story» вместо цифрВместо сухого «87% совместимости» генерировать короткий нарратив через YandexGPT: «Вы оба ценят свободу и глубину, но Аня тянется к слиянию, а Максим — к дистанции. Это притяжение противоположностей по стилю привязанности, которое работает при одном условии: честные разговоры о пространстве». Такой текст люди скринят и пересылают друзьям — органический рост.

4. HD-транзит дня (daily hook)Рассчитывать текущие планетарные позиции (уже есть ephem в стеке) и генерировать персонализированный «гороскоп дня» специфичный для HD-типа пользователя. Например: «Сегодня Солнце активирует ворота 47. Для Генератора с undefined Heart это день, когда не стоит соглашаться на встречу из давления». Это создаёт ежедневный повод открывать приложение — критично для retention.

5. Анонимный инвайт «Кто-то хочет познакомиться»Пользователь видит в мэтчах человека с высокой совместимостью, но не хочет писать первым. Добавить кнопку «Дать знак анонимно» — партнёр получает уведомление: «Один из ваших высоко совместимых совпадений хочет познакомиться. Откройте чат?». Это снижает страх отказа и значительно повышает конверсию в диалог. Аналог «Super Like», но психологически умнее.

6. AI-антипаттерн трекер в чате (Premium)Необязательная функция — пользователь сам включает. AI анализирует его сторону переписки и тихо, без осуждения, показывает паттерны: «Ты написал первым 5 раз подряд. Это может быть тревожный стиль привязанности в действии. Дай пространство». Видит только сам пользователь. Это решает заявленную боль про «токсичные паттерны, которые повторяются» и создаёт огромную ценность для Premium.

7. Групповые «Совместимые пространства» (Community)Для пользователей со схожими HD-типами и ценностями создавать временные тематические групповые чаты (например, «Генераторы+Проекторы, ценящие глубину, Москва»). Это снижает давление one-on-one и создаёт ощущение комьюнити — ключевой тренд 2025. Работает как dating app + клуб по интересам.

8. Голосовая нотка профиля30-секундный голосовой ответ на вопрос агента вместо текста — опционально. Это убирает страх первого сообщения ещё раньше и решает проблему катфишинга. Голосовые данные также можно использовать для sentiment analysis (добавляет слой к психологическому профилю). Приложения с голосом показывают на 35% выше конверсию матч→встреча.
