# Инструкция: Как связать GitHub и Netlify для автоматических обновлений

Поскольку у вас проект (фронтенд и бэкенд) лежит в одной папке, в Netlify нужно указать **правильную базовую директорию**. Следуйте этому простому алгоритму один раз, и дальше всё будет обновляться автоматически при каждой отправке на GitHub!

## ШАГ 1. Отправляем свежий код на GitHub (в Терминале)

Прямо сейчас у вас на компьютере лежит куча новых классных обновлений (экран создания профиля с выбором из 50 городов, отвязка от "Ольги", свободный вход по любому номеру). Давайте отправим их на ваш GitHub.

В новом терминале (находясь в папке `d:\Vpoiske App`) введите эти команды по очереди:

```bash
git add .
git commit -m "Добавлен экран регистрации и выбор городов"
git push
```
*Если `git push` просит пароль или авторизацию — пройдите её в браузере.*

---

## ШАГ 2. Подключаем репозиторий к Netlify

1. Зайдите в браузер на сайт **[app.netlify.com](https://app.netlify.com/)**.
2. Слева в меню нажмите **Sites**, затем справа сверху кнопку **Add new site** -> **Import from an existing project**.
3. Выберите провайдера **GitHub**. (Netlify попросит вас авторизовать приложение — нажмите Authorize).
4. Найдите поиском и выберите ваш репозиторий: `sannytg01-web/Vpoiske-App`.

---

## ШАГ 3. Настраиваем пути сборки (САМОЕ ВАЖНОЕ!)

На следующем экране Netlify покажет "Site settings". У вас сложный проект (внутри есть папки `frontend` и `backend`), поэтому Netlify нужно явно ткнуть носом в папку фронтенда.

Заполните поля **СТРОГО ТАК**:

* **Base directory**: `frontend`
* **Build command**: `npm install && npm run build`
* **Publish directory**: `frontend/dist` (иногда Netlify сам заменяет это на просто `dist` после установки Base directory — это тоже нормально, главное чтобы было слово dist).

Нажмите **Deploy site**.

---

## ШАГ 4. Готово! Авто-обновления настроены

Сам деплой займет около одной минуты (Netlify скачает ваш код с GitHub, установит зависимости и соберёт проект).
Когда сборка будет завершена, он выдаст вам зеленую ссылку (типа `https://vlubvi-super-app.netlify.app`).

**Что делать дальше?**
1. Откройте `backend/.env` и пропишите туда вашу новую постоянную ссылку от Netlify.
2. Перезапустите `bot.py` один раз.
3. **ВСЁ!** Теперь каждый раз, когда мы с вами будем программировать новый функционал (например, подключать СМС-шлюз или менять дизайн), вам нужно будет просто сделать `git add .`, `git commit -m "апдейт"` и `git push`.
Netlify **сам** увидит изменения на GitHub и за пару минут без вашего участия обновит приложение у всех пользователей!
